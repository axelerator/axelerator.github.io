    Objektorientierte Programmierung mit Java

    Im Sommersemester 2015 wurde die OOP Vorlesung nach den OOP Folien von Herrn Schmidt gehalten.

    Am Ende des Semesters wurde folgende Klausur gestellt: oop_ss_2015.pdf

    Zum Ende des WS 15/16 wurde die folgende Klausur gestellt:oop_klausur_ws1516.pdf

    Da die Vorlesung so nicht mehr stattfindet, habe ich mich bereit erklärt ein weiters mal eine Klausr zu stellen.

    • Die Klausur wird eine ähnliche Struktur wie die beiden anderen Klausur haben
    • Stellen Sie sicher dass Sie die Sprachmittel zur Implementierung ganzer Klassen beherrschen (nicht nur einzelner Methodenrümpfe)
    • Das Verständnis vom Verhältnis von Objekten zu Referenzen wird vorrausgesetzt
    • Die in der Vorlesung behandelten Sprachmittel und Laufzeitfunktionen der Java Programmiersprache werden geprüft werden.
    • Das Verständnis der Prinzipien der objektorientieren Programmierung wird durch Programmieraufgaben geprüft werden
    • Es wird keine Fragen zu Threads/Nebenläufigkeit geben
